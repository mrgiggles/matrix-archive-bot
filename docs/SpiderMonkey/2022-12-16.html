<iain>
You can throw any value you want in JS (eg throw 3). GetPendingExceptionStack returns the pending exception, if there is one, and may have an associated SavedFrame object. ExceptionStackOrNull takes an ErrorObject, which is thrown by the engine in a variety of cases, and extracts the stack property from that object if it is present.
